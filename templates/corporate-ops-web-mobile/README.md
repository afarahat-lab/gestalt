# Corporate Operations Web & Mobile — Tier 1 Template

**Tier:** 1 (Standard Library — ships with AgentForge SDLC)
**Version:** 0.1.0
**Type:** harness-template

A complete harness starter for corporate operations web and mobile applications.
Covers the patterns and constraints common to HR, Finance, Procurement, and other
internal operations platforms.

---

## What this template provides

- **Modular monolith architecture** with enforced module boundaries
- **RBAC framework** — role-based access control at middleware, not inline
- **Audit trail** — immutable operation log, enforced on all state-changing routes
- **Approval workflow primitive** — state machine for multi-step approvals
- **Corporate identity** — SSO via SAML/OIDC/Kerberos (no local passwords)
- **Domain model primitives** — User, Organisation, Role, AuditLog, Notification, WorkflowInstance
- **Constraint rules** — 11 architectural rules pre-loaded and enforced from day one
- **Golden principles** — 7 non-negotiables for enterprise operations software

## Who should use this template

Projects that are building:
- HR self-service portals (leave, payroll, org chart)
- Finance operations tools (expense approval, budget tracking)
- Procurement platforms (purchase orders, vendor management)
- Any internal operations app requiring RBAC, audit trails, and SSO

---

## Domain coverage

This template is intentionally domain-agnostic within the corporate operations
space. The harness provides the cross-cutting concerns (auth, audit, RBAC, workflows).
Your business domain entities are generated by the harness initializer based on
your project description.

---

## Compliance notes

The golden principles in this template satisfy the common requirements of:
- **SOC 2 Type II** — audit trails, access controls, change management
- **GDPR** — data access logging, no sensitive data in logs
- **ISO 27001** — access control, audit logging, change records

For domain-specific compliance (HIPAA, PCI-DSS, local regulations),
look for specialised Tier 2/3 templates in the registry that extend this one.
